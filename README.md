# @bfehub/lint

用于快速为项目添加代码规范和提交规范。也作为现代规范的基础范例和自定义需求的基础配置。

```sh
# use npm
npm create @bfehub/lint
# use yarn
yarn create @bfehub/lint
# use pnpm
pnpm create @bfehub/lint
# when pnpm workspace
pnpm create @bfehub/lint -w
```
